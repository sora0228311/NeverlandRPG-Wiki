# 統合版(Bedrock Edition)をアップデートした後にサーバーに参加できない

プラグインのverが合わないことが原因です。
対応できる方がオーナーさんのみのため、対応を待ちましょう。


# サーバーに接続できないんだけど...

NeverlandRPGサーバーは奇数の時間に再起動をします。
再起動後は参加する前に、3分程開けてから試してみましょう。
それでも参加できない、という場合はサーバーがダウンしている可能性があります。その際は、
<#i-server-info> を見ることでサーバーの起動状況を確認できます。

「🛑 **サーバーが停止しました**」
というメッセージから動きがない！

という時はダウンしていますので、<#-報告・通報＞鯖落ち報告>にて「サーバー落ちました！」等投稿して頂けると助かります。
(一人でも報告してたら書かなくてokです)

# 役職がニートになり、進行度がリセットされてしまった

ニートバグと呼ばれるものです。対処法としては
・サーバーの再起動を待つ
・運営さんに対応してもらう
の2つです。
