## あらすじ
Neverland RPG Season１[消え逝く世界]

平穏な日々が続く[フェッシャラ地方]
この地方は旅人たちが最初に通る地方として知られている
そんな地方である日を境に魔物が狂暴化し被害が相次ぐようになった
各地のエリアで同じ事例が増え困り果てた時
冒険者が手を差し伸べた。
だが、この時、誰も世界が崩壊へ向かっているとは思わなかった・・・
さぁ、今冒険へ出発し、狂暴化の原因を突き止めよ！
